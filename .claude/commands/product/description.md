Write a product description for an online shop. The product is: $ARGUMENTS

## Target Customer

The target customer is defined in @context/marketing-seo/avatar-spa.md. Tailor the message to directly address this customer's pain points and personal goals.

## Features & Benefits

The product features can be found in @context/marketing-seo/features-formula-8-spa.md. Translate these features into clear benefits that answer the customer's question: "What's in it for me?"

- Instead of just stating a feature, explain the tangible benefit (e.g., instead of "nylon weave backrest", say "flexible support for good posture and all-day comfort")
- Focus on telling a story that shows why this is the perfect solution

## Tone of Voice

Write using the tone of voice from @context/marketing-seo/tono-de-voz.md.

- Avoid exaggerations and over-the-top language
- Keep the text honest, transparent, and trustworthy

## SEO Keywords

Include keywords naturally from @context/marketing-seo/keywords-aceite-masaje-spa.md.

- Avoid keyword stuffing — the text must remain engaging and readable for humans

## Formatting

Make the text highly skimmable:

- Break long paragraphs into shorter sections
- Use bullet points to highlight the most important features and benefits
- Use clear headings
- Emphasize key points using **bold** or *italic* text
